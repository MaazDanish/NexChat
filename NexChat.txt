NexChat: Revolutionizing Communication

Welcome to NexChat, where seamless communication meets unparalleled user experience! NexChat is a cutting-edge web application that redefines the way we connect and interact online. With its sleek design, intuitive interface, and robust features, NexChat empowers users to engage in real-time conversations effortlessly.

Features
Real-time Chatting: Enjoy seamless real-time messaging with friends and groups.
Image Sharing: Share images instantly within chats to enhance communication.
Forgot Password: Easily reset your password using OTP sent to your registered email or phone number.
User Profile Management: Update user details and profile pictures with ease.
Group Creation and Management: Create and manage groups, set group settings, and invite users to join.
User Information Display: View user profiles and information within the app.

Installation
To run NexChat locally, follow these steps:

Clone this repository.
Install dependencies using npm install.
Set up your database and environment variables.
Run the application using npm start.
